# Pediatric-clinic-management-system
This is a menu-driven console application written in C for managing a pediatric clinic.
The program allows users to enter child and parent details, book appointments for different clinic services, and store records using file handling.
## Features
- Menu-driven pediatric clinic system
- Vaccination scheduling based on age group
- Appointment booking with day and date
- Prevents double booking of slots
- Stores records using text file handling
- View and delete appointments
## Files Used
- clinic.c   → Main source code
- clinic.txt → Stores appointment and patient details
## Learning Outcomes
- Understanding of structures in C
- Practical use of file handling
- Menu-driven program design
- Appointment management logic
